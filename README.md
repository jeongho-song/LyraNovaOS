# LyraNovaOS

LyraNovaOS is an OS for autonomous systems that operate and evolve together with humans and AI.

## Vision
- Sustainability over ownership
- Sharing over monopoly
- Citizen over individual

## Status
- Early stage (v0.1)
- Under active development

## Structure
- LyraNovaOS is not tied to any specific company,
and is scheduled to transition to a Foundation structure in the future.
